## 主要コンポーネント

### フロントエンド
- **フレームワーク**: React + TypeScript
- **ビルドツール**: Vite
- **UIライブラリ**: Tailwind CSS
- **ルーティング**: React Router

### バックエンド
- **ランタイム**: Node.js 20
- **フレームワーク**: Express.js
- **ORM**: Prisma 7.4.0
- **認証**: JWT (jsonwebtoken)

### データベース
- **DBMS**: PostgreSQL 16
- **永続化**: Docker Volume (pgdata)

### インフラ
- **クラウド**: AWS EC2 (Amazon Linux 2023)
- **リージョン**: ap-northeast-1 (東京)
- **コンテナ**: Docker Compose
- **リバースプロキシ**: Cloudflare Tunnel


## デプロイ手順

1. ローカルでコード変更
2. EC2にファイル転送 (scp)
3. Docker Compose でビルド・再起動
   ```bash
   docker-compose -f docker-compose.prod.yml build --no-cache
   docker-compose -f docker-compose.prod.yml up -d
   ```

## 重要なファイルパス

### EC2
- アプリケーション: `/home/ec2-user/app/`
- Cloudflare設定: `/etc/cloudflared/config.yml`
