## 主要コンポーネント

### フロントエンド
- **フレームワーク**: React + TypeScript
- **ビルドツール**: Vite
- **UIライブラリ**: Tailwind CSS
- **ルーティング**: React Router

### バックエンド
- **ランタイム**: Node.js 20
- **フレームワーク**: Express.js
- **ORM**: Prisma 7.4.0
- **認証**: JWT (jsonwebtoken)

### データベース
- **DBMS**: PostgreSQL 16
- **永続化**: Docker Volume (pgdata)

### インフラ
- **クラウド**: AWS EC2 (Amazon Linux 2023)
- **リージョン**: ap-northeast-1 (東京)
- **コンテナ**: Docker Compose
- **リバースプロキシ**: Cloudflare Tunnel

## デプロイ手順

### 方法1: 手動デプロイ（推奨）
1. ローカルでコード変更・コミット
   ```bash
   git add .
   git commit -m "変更内容"
   git push origin main
   ```

2. サーバーファイルのパッケージ作成
   ```powershell
   # deploy-temp ディレクトリ作成
   New-Item -ItemType Directory -Path deploy-temp
   
   # 必要なファイルをコピー
   Copy-Item -Path "server/src" -Destination "deploy-temp/server/src" -Recurse
   Copy-Item -Path "server/package.json" -Destination "deploy-temp/server/"
   Copy-Item -Path "server/Dockerfile" -Destination "deploy-temp/server/"
   Copy-Item -Path "server/prisma" -Destination "deploy-temp/server/prisma" -Recurse
   
   # アーカイブ作成
   tar -czf deploy-package.tar.gz -C deploy-temp .
   ```

3. EC2にアップロード
   ```powershell
   scp -i "$env:USERPROFILE\snsinsight-key2.pem" deploy-package.tar.gz ec2-user@54.168.247.161:~/
   ```

4. EC2でデプロイ実行
   ```bash
   ssh -i "$env:USERPROFILE\snsinsight-key2.pem" ec2-user@54.168.247.161
   
   # ファイル展開
   cd /home/ec2-user/app
   tar -xzf ~/deploy-package.tar.gz -C server/
   rm ~/deploy-package.tar.gz
   
   # サーバーコンテナ再ビルド・再起動
   docker stop snsinsight-server
   docker rm snsinsight-server
   docker build -t app-server ./server
   
   # 環境変数を設定して起動
   docker run -d --name snsinsight-server \
     --network app_app-network \
     -e PORT=3001 \
     -e DATABASE_URL='postgresql://postgres:postgres@snsinsight-postgres:5432/snsinsight' \
     -e NODE_ENV=production \
     -e JWT_SECRET='***' \
     -e META_APP_ID='***' \
     -e META_APP_SECRET='***' \
     -e META_REDIRECT_URI='https://app.trendio.jp/api/auth/meta/callback' \
     -e TIKTOK_CLIENT_KEY='***' \
     -e TIKTOK_CLIENT_SECRET='***' \
     -e TIKTOK_REDIRECT_URI='https://app.trendio.jp/api/auth/tiktok/callback' \
     -e XAI_API_KEY='***' \
     -e XAI_MODEL='grok-4-1-fast-non-reasoning' \
     app-server
   
   # 確認
   docker ps
   docker logs snsinsight-server
   ```

## トラブルシューティング

### 502 Bad Gateway エラー
- **原因**: サーバーコンテナのポート設定ミス
- **確認**: `docker logs snsinsight-server` でポート番号を確認
- **解決**: 環境変数 `PORT=3001` を設定してコンテナを再起動

### コンテナが起動しない
```bash
# コンテナ状態確認
docker ps -a

# ログ確認
docker logs snsinsight-server
docker logs snsinsight-frontend
docker logs snsinsight-postgres

# ネットワーク確認
docker network ls
docker network inspect app_app-network
```

### データベース接続エラー
- PostgreSQLコンテナが起動しているか確認
- DATABASE_URL環境変数が正しいか確認
- ネットワーク接続を確認
