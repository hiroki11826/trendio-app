# Facebookページ作成手順（山田紘丘アカウント）

## 現在の状況

ログ: `User info: { id: '122097920955261133', name: '山田紘丘' }`
問題: このアカウントが管理しているFacebookページが0個

## 解決策: 新しいページを作成

### ステップ1: Facebookでページを作成

1. **Facebookにログイン**
   - アカウント: 山田紘丘
   - [https://www.facebook.com/](https://www.facebook.com/)

2. **左サイドバーの「ページ」をクリック**

3. **「新しいページを作成」ボタンをクリック**

4. **ページ情報を入力**:
   - **ページ名**: `BuzzInsight`
   - **カテゴリ**: `ビジネス・経済` または `ソフトウェア会社`
   - **説明**: `SNS運用管理ツール`

5. **「ページを作成」をクリック**

### ステップ2: Instagramアカウントをリンク

1. **作成したBuzzInsightページを開く**

2. **左サイドバーの「設定」をクリック**

3. **「Instagram」セクションを探す**
   - 見つからない場合は、検索ボックスで「Instagram」を検索

4. **「アカウントをリンク」をクリック**

5. **Instagramにログイン**
   - あなたのInstagramアカウントでログイン

6. **ビジネスアカウントに切り替え**（まだの場合）
   - Instagramアプリで:
   - プロフィール → メニュー（三本線）
   - 「設定とプライバシー」
   - 「アカウントタイプとツール」
   - 「プロフェッショナルアカウントに切り替える」
   - 「ビジネス」を選択

7. **リンクを承認**

### ステップ3: ページの管理者権限を確認

1. **BuzzInsightページの「設定」を開く**

2. **左サイドバーの「ページの役割」をクリック**

3. **あなた（山田紘丘）が「管理者」になっているか確認**

### ステップ4: 接続テスト

1. **サーバーを再起動**（既に起動している場合）
   ```powershell
   # Ctrl+Cで停止
   npm run dev
   ```

2. **ブラウザで接続テスト**
   - `http://localhost:3000/settings`にアクセス
   - 「Instagram」の「接続する」をクリック
   - 山田紘丘アカウントでログイン
   - 権限を承認

3. **サーバーログを確認**
   ```
   === Meta API Response ===
   Total pages found: 1  ← これが1になればOK
   Page 1: {
     id: '...',
     name: 'BuzzInsight',
     has_instagram: true,
     instagram_id: '...'
   }
   ========================
   ```

## 代替案: Meta Appに権限を追加

もしページが既に存在するのに表示されない場合、Meta Appに権限を追加する必要があります。

### Graph API Explorerで確認

1. **[Graph API Explorer](https://developers.facebook.com/tools/explorer/)にアクセス**

2. **右上で「Meta App」を選択**
   - あなたのアプリ: 1606160517196384

3. **「アクセストークンを取得」をクリック**

4. **権限を選択**:
   - `pages_show_list` ← これが重要
   - `pages_read_engagement`
   - `instagram_basic`
   - `instagram_manage_insights`

5. **「アクセストークンを生成」をクリック**

6. **クエリフィールドに入力**:
   ```
   me/accounts?fields=id,name,instagram_business_account
   ```

7. **「送信」をクリック**

8. **レスポンスを確認**:
   - ページが表示される → 権限OK、アプリの設定を確認
   - 空の配列 → ページが存在しないか、権限がない
   - エラー → 権限が不足

## Meta Appに権限を追加する方法

### 方法1: Facebookログイン製品を追加

1. **Meta for Developersダッシュボード**
   - [https://developers.facebook.com/apps/1606160517196384](https://developers.facebook.com/apps/1606160517196384)

2. **「製品を追加」セクションを探す**

3. **「Facebookログイン」を見つける**

4. **「設定」ボタンをクリック**

5. **設定を完了**
   - プラットフォーム: ウェブ
   - サイトURL: `http://localhost:3000`
   - 有効なOAuthリダイレクトURI: `http://localhost:3000/auth/meta/callback`

### 方法2: アプリレビューで権限をリクエスト

1. **左サイドバーで「アプリレビュー」をクリック**

2. **「権限とフィーチャー」タブをクリック**

3. **検索ボックスで`pages_show_list`を検索**

4. **「リクエスト」または「追加」をクリック**

5. **同様に以下も追加**:
   - `pages_read_engagement`
   - `instagram_basic`
   - `instagram_manage_insights`

## トラブルシューティング

### Q: Facebookで「ページ」が見つからない

**A**: 左サイドバーの「すべて表示」をクリックして、「ページ」を探してください。

### Q: Instagramセクションが見つからない

**A**: ページの設定で検索ボックスを使って「Instagram」を検索してください。

### Q: ビジネスアカウントに切り替えられない

**A**: Instagramアプリ（モバイル）で切り替える必要があります。ウェブ版では切り替えできません。

### Q: リンクを承認できない

**A**: Instagramアカウントが既に別のFacebookページにリンクされている可能性があります。先に解除してください。
